PetGuard Pro

An intelligent, low-cost pet safety system combining an onboard telemetry collar, Firebase cloud syncing, and a feature-rich Flutter app with safe-zone trackers, vitals streams, and PDF behavior reports.

01 · Problem Statement

The Gap in Pet Monitoring

Conventional pet trackers report location on a delay and say nothing about how the pet is actually doing. PetGuard Pro closes both gaps with a connected collar and a live dashboard.

Latency gaps

Slow, Infrequent updates

Many consumer collars refresh location every few minutes, leaving owners blind during the moments that matter most. PetGuard Pro's ESP32 collar pushes sensor readings to Firebase Realtime Database on a short, fixed interval, so the app reflects the pet's current state within seconds, not minutes.

Delayed response

No Way to Intervene Remotely

Most trackers only notify after a pet is already gone. When a geofence boundary is crossed, PetGuard Pro triggers an onboard piezo buzzer on the collar itself, giving the owner an immediate, local deterrent rather than just a phone alert after the fact.

Inadequate context

No Health Insights

Location-only devices say nothing about how a pet is actually doing. Our collar pairs an MLX90614 infrared sensor for surface temperature with motion data for real-time activity level — giving owners health context, not just a dot on a map.

Stale dashboards

Silent Disconnects

If a collar loses power or signal, most apps keep showing the last value with no warning that it's outdated. PetGuard Pro's app checks the timestamp on each incoming Firebase record and flags the pet's status as offline the moment updates stop arriving.

⚡ Onboard Dual Connection

Collar automatically synchronizes local telemetry data packet frames using home Wi-Fi or fallbacks onto SIM800L cellular lines when walking outside.

💓 Real-Time Vitals Tracking

Direct registers readings map optical heart rates and IR surface temperatures. Triggers notifications if indicators pass safety limits.

👟 Accelerometer Step Counting

Translates 3-axis MPU6050 accelerometer ticks directly into live step counting goals, monitoring active play metrics on the dashboard.

🚧 Dynamic Virtual Fence

Compares incoming coordinates against boundary circles. Escalates alerts and triggers hardware buzzer alarms if the perimeter is crossed.

📊 Behavior Analytics PDF Reports

Compiles today's path coordinates records, average vitals levels, and step counts into an exportable PDF behavioral summary report.

🎫 Support Tickets Desk

If collar connections drop, status updates to offline. The built-in desk logs tickets in Firestore to handle issues directly.
